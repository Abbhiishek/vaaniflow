Settings

Transcription server

Your LocalAI / OpenAI-compatible Whisper endpoint (e.g. the service in your AKS cluster).

AI polish

A chat model cleans transcripts: filler words out, self-corrections resolved, punctuation fixed. Leave the endpoint empty to use the Whisper server above, or point it at any OpenAI-compatible provider (a fast one like Groq or Cerebras keeps polish under a second). If the deadline passes, the raw transcript is pasted instead, so dictation never waits on a slow model. Writing style lives on the Style page.
