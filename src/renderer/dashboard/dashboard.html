Style

How the AI writes your speech — needs a chat model set in Settings → AI polish.

Writing style

Detects where the text will land and writes accordingly: email apps get professional prose, chat apps get relaxed messages, AI assistants get precise prompts, code editors and terminals keep technical terms exact, and document apps get polished writing. Your per-app rules below always win.

Free-form guidance given to the AI on every dictation.

Tone per app

Override the tone based on the app you're dictating into. First match wins, including over automatic adaptation.

Snippets

Say a trigger phrase (or "insert" + the phrase) as the whole dictation and the snippet text is inserted instead.

Settings

Azure OpenAI

VaaniFlow reads the Azure endpoint, API key, Whisper deployment, and polish deployment from a local configuration file. The file is reloaded before each dictation.

Configuration file Loading… 
{
  "baseUrl": "https://your-resource.openai.azure.com",
  "apiKey": "your-api-key",
  "apiVersion": "2024-10-21",
  "whisperDeployment": "whisper",
  "llmDeployment": "gpt40"
}

AI polish

Transcript cleanup uses llmDeployment from config.json on the same Azure endpoint and API key. Leave it empty to disable the LLM stage. If the deadline passes, the raw transcript is pasted instead.
